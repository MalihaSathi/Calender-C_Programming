# Calender-C_Programming
This program is a simple calendar generator written in the C programming language that displays the complete calendar for a given year. The user enters a year, and the program calculates the first day of that year using the getFirstDayoftheYear() function. It stores the names of the months and the number of days in each month using arrays. The program also checks whether the entered year is a leap year and updates February to 29 days if necessary. Using loops and formatted printing, it displays all twelve months one by one in a calendar format with proper alignment of weekdays and dates. The program demonstrates important programming concepts such as functions, arrays, loops, conditional statements, leap year calculation, and formatted output in C.
